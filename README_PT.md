# ☕ MOOC.fi Programação Java — Soluções dos Exercícios

Este repositório contém minhas soluções para o curso [Java Programming I & II](https://java-programming.mooc.fi/) oferecido pela **Universidade de Helsinque**.

## 📚 Sobre o Curso

Um dos cursos gratuitos de Java mais completos disponíveis, utilizado para ensinar estudantes de Ciência da Computação na Universidade de Helsinque. Cobre desde fundamentos até orientação a objetos, estruturas de dados, streams, I/O de arquivos e JavaFX.

| Curso | Parts | Exercícios |
|---|---|---|
| Java Programming I | Parts 1–7 | ~50 exercícios |
| Java Programming II | Parts 8–14 | ~56 exercícios |

---

## 🗂️ Java Programming I

### Part 1 — Fundamentos
- Impressão de texto com `System.out.println()`
- Variáveis e tipos primitivos: `int`, `double`, `String`, `boolean`
- Leitura de entrada do usuário com `Scanner`
- Concatenação de strings
- Conversão de tipos: `Integer.parseInt()`, `Double.parseDouble()`, `Boolean.parseBoolean()`

### Part 2 — Repetição e Métodos
- Loops `while` e `for`
- Operações e métodos de String
- Criação e chamada de métodos próprios
- Parâmetros e retorno de métodos

### Part 3 — Listas e Arrays
- `ArrayList` e arrays
- Iteração sobre listas com loops
- Busca e ordenação de elementos em listas

### Part 4 — Orientação a Objetos (Introdução)
- Criação de classes e objetos
- Construtores, atributos e métodos de instância
- Encapsulamento e modificadores de acesso

### Part 5 — OO Avançado
- Herança com `extends`
- Interfaces e polimorfismo
- Classes abstratas
- Sobrescrita de `toString()` e `equals()`
- Enums

### Part 6 — Agrupamento de Dados
- `HashMap` e `HashSet`
- Iteradores
- Organização de dados complexos com coleções

### Part 7 — Programação em Larga Escala
- Separação de responsabilidades e pacotes
- Algoritmos de busca e ordenação
- Expressões regulares
- Introdução a testes unitários com JUnit

---

## 🗂️ Java Programming II

### Part 8 — HashMap e Recursão
- Uso avançado de `HashMap`
- Conjuntos e iteradores
- Algoritmos recursivos

### Part 9 — Interfaces e Herança (Avançado)
- Herança avançada
- Interfaces funcionais
- Polimorfismo em profundidade

### Part 10 — Tratamento de Exceções
- `try`, `catch`, `finally`
- Exceções customizadas
- Pilha de chamadas

### Part 11 — Streams e Lambdas
- Stream API: `filter`, `map`, `sorted`, `collect`
- Expressões lambda
- `Optional`

### Part 12 — I/O de Arquivos
- Leitura e escrita de arquivos
- `BufferedReader` e `BufferedWriter`
- Serialização

### Part 13 — JavaFX
- Interface gráfica com JavaFX
- Layouts, eventos e componentes visuais
- FXML

### Part 14 — Algoritmos e Estruturas de Dados
- Estruturas de dados avançadas
- Algoritmos de ordenação
- Análise de complexidade
- Testes unitários avançados com JUnit
