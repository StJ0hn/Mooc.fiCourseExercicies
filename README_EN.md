# ☕ MOOC.fi Java Programming — Exercise Solutions

This repository contains my solutions to the [Java Programming I & II](https://java-programming.mooc.fi/) course offered by the **University of Helsinki**.

## 📚 About the Course

One of the most complete free Java courses available, used to teach Computer Science students at the University of Helsinki. It covers everything from fundamentals to object-oriented programming, data structures, streams, file I/O, and JavaFX.

| Course | Parts | Exercises |
|---|---|---|
| Java Programming I | Parts 1–7 | ~50 exercises |
| Java Programming II | Parts 8–14 | ~56 exercises |

---

## 🗂️ Java Programming I

### Part 1 — Fundamentals
- Printing text with `System.out.println()`
- Variables and primitive types: `int`, `double`, `String`, `boolean`
- Reading user input with `Scanner`
- Basic string concatenation
- Type conversion: `Integer.parseInt()`, `Double.parseDouble()`, `Boolean.parseBoolean()`

### Part 2 — Repetition and Methods
- `while` and `for` loops
- String operations and methods
- Creating and calling custom methods
- Method parameters and return values

### Part 3 — Lists and Arrays
- `ArrayList` and arrays
- Iterating over lists with loops
- Searching and sorting elements in lists

### Part 4 — Object-Oriented Programming (Introduction)
- Creating classes and objects
- Constructors, attributes and instance methods
- Encapsulation and access modifiers

### Part 5 — OOP Advanced
- Inheritance with `extends`
- Interfaces and polymorphism
- Abstract classes
- Overriding `toString()` and `equals()`
- Enums

### Part 6 — Data Grouping
- `HashMap` and `HashSet`
- Iterators
- Organizing complex data with collections

### Part 7 — Large-scale Programming
- Separation of concerns and packages
- Search and sorting algorithms
- Regular expressions
- Introduction to unit testing with JUnit

---

## 🗂️ Java Programming II

### Part 8 — HashMap and Recursion
- Advanced `HashMap` usage
- Sets and iterators
- Recursive algorithms

### Part 9 — Interfaces and Inheritance (Advanced)
- Advanced inheritance
- Functional interfaces
- Polymorphism in depth

### Part 10 — Exception Handling
- `try`, `catch`, `finally`
- Custom exceptions
- Call stack

### Part 11 — Streams and Lambdas
- Stream API: `filter`, `map`, `sorted`, `collect`
- Lambda expressions
- `Optional`

### Part 12 — File I/O
- Reading and writing files
- `BufferedReader` and `BufferedWriter`
- Serialization

### Part 13 — JavaFX
- Graphical user interface with JavaFX
- Layouts, events and visual components
- FXML

### Part 14 — Algorithms and Data Structures
- Advanced data structures
- Sorting algorithms
- Complexity analysis
- Advanced unit testing with JUnit
